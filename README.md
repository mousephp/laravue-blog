## Installation

- *cd laravue-blog*
- *composer install*
- Copy .env.example to .env:<br/>
 Windows: *copy .env.example .env<br/>*
 Ubuntu: *cp .env.example .env<br/>*
- Run: *php artisan key:generate*
- Create database and editting .env
- Run: *php artisan migrate --seed*
- Run: *php artisan serve*
- Open new terminal:
- Run: *npm install*
- Run: *npm run watch*
